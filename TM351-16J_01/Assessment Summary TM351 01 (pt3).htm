Qu2.
You have done a very good job using OpenRefine to deal with missing and odd data. That said, there were still schools with no parish in the dataframe. I found a webite that returns a parish for any postcode... https://www.achurchnearyou.com/parishmap.php

I must admit that when using such trivially small datasets such as those in the TMA, I find it more useful to create new dataframes (with unique names) throughout the slicing and dicing process. It makes debugging easier should there be problems.
Also, due to the samll size, there is little need to reduce the columns available to you too early in the data analysis process. When you have many MB of data, it may be worth doing, but these files are so small.
